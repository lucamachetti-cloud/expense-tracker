# Expense Tracker

A feature-rich web application to track your personal expenses with budgeting, categorization, and visualizations.

## Features

- **Expense Management**: Add, edit, and delete expenses with description, amount, category, and date
- **Monthly Budget**: Set and track your overall monthly budget
- **Category Budgets**: Set individual budgets for 7 different categories
- **Visual Progress Tracking**: Color-coded progress bars showing spending vs budget
- **Budget Alerts**: Get warnings when approaching or exceeding budgets (80% and 100%)
- **Filtering**: Filter expenses by date range and category
- **Charts**: Interactive pie chart and line chart showing spending patterns
- **Data Export**: Export your data as CSV or JSON
- **Data Persistence**: All data stored locally in your browser (localStorage)

## Categories

- Food & Dining
- Transportation
- Shopping
- Entertainment
- Bills & Utilities
- Healthcare
- Other

## How to Use Locally

Simply open `index.html` in your web browser.

## Deployment Options

### Option 1: GitHub Pages (Recommended - Free)

1. Create a GitHub account at https://github.com
2. Create a new repository (e.g., "expense-tracker")
3. Upload all files to the repository
4. Go to Settings > Pages
5. Select "Deploy from a branch" and choose "main" branch
6. Your site will be live at `https://yourusername.github.io/expense-tracker`

### Option 2: Netlify (Free)

1. Create account at https://netlify.com
2. Drag and drop the entire expense-tracker folder
3. Your site will be live instantly with a custom URL
4. Optional: Set up custom domain

### Option 3: Vercel (Free)

1. Create account at https://vercel.com
2. Import your GitHub repository or drag and drop files
3. Deploy with one click
4. Your site will be live with a custom URL

### Option 4: Cloudflare Pages (Free)

1. Create account at https://pages.cloudflare.com
2. Connect your GitHub repository
3. Deploy automatically
4. Free with unlimited bandwidth

## Technologies Used

- HTML5

## Privacy

All your expense data is stored locally in your browser. No data is sent to any server. Your financial information stays completely private on your device.
